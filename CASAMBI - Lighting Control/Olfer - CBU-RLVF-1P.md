# Olfer CBU-RLVF-1P - 1-Channel Voltage-Free Relay

## Đặc điểm

✅ **Voltage-Free Contact** - Dry contact, điều khiển bất kỳ loại tải/hệ thống

✅ **3 chế độ nguồn** - AC 110-240V, DC 24V, hoặc DC 12V (chọn 1)

## Voltage-Free Contact (Dry Contact)

**Chính khác biệt quan trọng:**
- **Switched-live relay** (CBU-2RL-2P): Cắt/ghép mains phase → chỉ AC on/off
- **Voltage-free relay** (CBU-RLVF-1P): Tiếp điểm neutral → điều khiển **bất kỳ** loại tải

**Ứng dụng:**
- Điều khiển garage door opener (thường 24VDC)
- HVAC system (24V transformer control)
- Valve control (AC/DC mix)
- Gate motor (12-24VDC)
- Relay trigger (bất kỳ voltage)

```

### Voltage-Free Contact Output

```
CBU-RLVF-1P (VF1 + VF2)
    ↓

(Bất kỳ loại điện áp/tải)

### Push Button Connection

```
Push Button (Normally Open)
    ↓
  CBU-RLVF-1P PUSH terminals (max 100m)
    ↓
  Triggers Casambi control
```

## Lắp đặt & Cài đặt

### Bước 1: Chọn & kết nối nguồn

1. **AC Mode:** 110-240VAC → input
2. **DC 24V:** 24VDC → input
3. **DC 12V:** 12VDC → input
(Chọn 1, không chọn nhiều)

### Bước 2: Kết nối tải

1. VF1 + VF2 → garage door/HVAC/valve/motor (dry contact)
2. (Optional) Push button → PUSH terminals

### Bước 3: Pair & cấu hình

1. Casambi App → add device
2. Chọn profile (default: Toggle)
3. Test: App toggle → tải bật/tắt

## Lưu ý

⚠️ **Voltage-free ≠ Switched-live** - Dry contact, không cắt mains

⚠️ **Không dùng metal case** - Chặn Bluetooth

⚠️ **Push button: Normally Open chỉ** - Không kết nối live parts

⚠️ **Chọn 1 chế độ nguồn** - AC hoặc DC 24V hoặc DC 12V (không mix)

⚠️ **Max 100m dây button** - Shielded cable nếu EMI mạnh
