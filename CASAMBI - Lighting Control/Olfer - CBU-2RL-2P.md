## Sơ đồ nối dây

### AC Mode (110-240VAC)

```
AC Power Supply
    ↓
  CBU-2RL-2P
  ├─ L (input phase)
  ├─ Output 1 (normally open relay) → Load 1
  ├─ Output 2 (normally open relay) → Load 2
  └─ N (neutral, if needed)
```

### DC Mode (24VDC)

```
24VDC Power Supply
    ↓
  CBU-2RL-2P
  ├─ + (positive)
  ├─ Output 1 (relay) → Load 1
  ├─ Output 2 (relay) → Load 2
  └─ - (negative)
```

### Push Button Connection

```
PB1 / PB2 (Normally Open Push Button)
    ↓
  CBU-2RL-2P (max 100m distance)
    ↓
  Triggers Casambi control
```

## Lắp đặt & Cài đặt

### Bước 1: Kết nối điện

1. **AC Mode:** Kết nối phase → input L
2. **DC Mode:** Kết nối (+) → input, (-) → ground
3. Kết nối tải vào Output 1/2
4. (Optional) Kết nối push button vào PB1/PB2

### Bước 2: Pair với Casambi

1. Mở Casambi App
2. Thêm device mới (gần relay)
3. Chọn profile (default: Toggle)
4. Cấu hình độc lập cho mỗi output nếu cần

### Bước 3: Test

- App: Toggle mỗi relay
- Push button: Kiểm tra trigger
- Tải: Xác nhận on/off hoạt động
