# DMX Engineering SceneDMXcas - DMX to Casambi Scene Selector

## Đặc điểm

✅ **DMX Scene Control** - Chọn 20 Casambi scenes qua DMX master

✅ **Intensity Control** - Channel 2 điều chỉnh 0-100% intensity

✅ **Casambi Long Range** - CBM003B radio (2.4GHz, up to 200ft outdoor)

✅ **12-24VDC** - Class 2 power, 30mA

✅ **Compact** - 72.6 x 30.0 x 18.0mm, 23g

## Cài đặt

1. **Power:** 12-24VDC Class 2 supply
2. **DMX:** Connect to DMX master (3-wire)
3. **Pair:** Add to Casambi network via App
4. **Config:** Set DMX Start Address + 20 scene mappings
5. **Control:** Use DMX master to select scene & intensity

## Lưu ý

⚠️ **Không dùng metal enclosure** - Chặn Bluetooth signal

⚠️ **Mesh repeater** - SceneDMXcas hoạt động như repeater trong mạng

- **Theater:** Control lighting scenes qua DMX master
- **Live Event:** Select pre-defined ambience scenes
- **Retail:** Switch lighting themes từ DMX console
